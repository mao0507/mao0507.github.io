簡介

在工作中常常會需要去接手他人的程式碼，每個專案所使用的開發環境不一定相同，用的node.js版本就有很多不同的版本，
這個時候就可以使用NVM來輕鬆切換不同版本的node.js，來解決不同版本所產生的問題。

NVM 為 Node Version Manager 的縮寫 ， 意思就是 Node.js的版本管理器

常使用到的功能 :

  • 切換node.js版本
  • 安裝不同版本的node.js作為環境

下載教學
